# LifeSpecs

LifeSpecs is an offline-first PWA cheat sheet for home and vehicle specs. It keeps homes and vehicles as first-class assets, lets vehicles link back to a home, and stores all data locally with export/import backup.

## Local Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run test
npm run build
```

The production build is configured for GitHub Pages at `/LifeSpecs/`.
